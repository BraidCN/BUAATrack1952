# 航迹1952 —— 北航校史智能向导小程序

北京航空航天大学校园历史文化微信小程序。以"AI+X"理念融合人工智能与校史文化传播，涵盖校史浏览、智能问答、每日推荐、趣味闯关等功能。

## 技术栈
- 微信小程序原生框架（WXML + WXSS + JS）
- 纯前端离线版，零后端依赖
- 72 条校史数据（1950-2025）
- 26 组关键词匹配 AI 问答引擎

## 版本
v0.1.3

## 部署
1. 下载[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 导入本项目目录
3. 填写 AppID 后即可编译预览
